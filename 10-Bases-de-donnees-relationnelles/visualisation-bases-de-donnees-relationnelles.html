Pr. EL HADIQ Zouhair

On suit l'ordre réel d'évaluation d'un SELECT : FROM/JOIN → WHERE → GROUP BY → agrégat → HAVING → SELECT → ORDER BY.

SELECT E.nom, ROUND(AVG(I.note),1) AS moyenneFROM Etudiant EJOIN Inscription I ON E.id = I.id_etudiantWHERE I.note >= 10GROUP BY E.nomHAVING AVG(I.note) >= 12ORDER BY moyenne DESC;
